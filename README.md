# 🇮🇳 Saarthi AI

### **Your AI Civic Companion**

**Making Government Services Simple with AI**

Saarthi AI is a **GenAI-powered civic assistance platform** built to simplify how citizens access government services across India. It combines **Artificial Intelligence, Interactive Maps, Smart Search, and Civic Assistance** into one modern platform, enabling users to quickly find government offices, understand document requirements, report civic issues, and discover eligible government schemes through an intelligent AI assistant.

---

## 🚀 Features

### 🤖 AI Civic Assistant
- AI-powered conversational assistant using Groq LLM
- Answers government-related queries
- Explains government procedures in simple language
- Provides structured responses instead of long paragraphs
- Supports English and Hindi

### 🗺️ Smart Government Services Map
- Interactive OpenStreetMap powered dashboard
- Search government services by city or category
- Category filters
- Auto zoom to search results
- AI explanation for every location
- Realistic demo dataset across major Indian cities

### 📄 AI Document Assistant
Get complete guidance for:
- Passport
- Aadhaar
- PAN Card
- Driving Licence
- Birth Certificate
- Income Certificate
- Ration Card

AI provides:
- Required Documents
- Application Steps
- Fees
- Processing Time
- Helpful Tips

### 📸 AI Issue Reporter
Report civic problems using AI.

Generate:
- Complaint Title
- Complaint Description
- Responsible Department
- Priority
- Issue Category

### 🎯 Government Scheme Recommender
Personalized scheme recommendations based on:
- Age
- State
- Income
- Occupation
- Student Status

Returns:
- Eligible Schemes
- Benefits
- Required Documents
- Application Process

### 🌐 Multilingual Support
- English
- Hindi

The entire interface changes dynamically based on the selected language.

### ♿ Accessibility
- Large Text Mode
- High Contrast Mode

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/arsh-siddiqui/Saarthi-AI.git
```

```bash
cd Saarthi-AI
```

### Install Dependencies

```bash
npm install
```

### Create Environment File

Create `.env.local`

```env
GROQ_API_KEY=YOUR_GROQ_API_KEY
```

### Start Development Server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
