# Saarthi AI – Smart Bharat Civic Companion

Saarthi AI is an intelligent, AI-powered civic companion designed to simplify government services for citizens across India. By offering a unified, multilingual platform, it helps users navigate complex processes, report civic issues, understand required documents, and discover eligible welfare schemes.

## Key Features

- **AI Assistant**: Natural language guidance for any government process.
- **Smart Maps**: Locate nearest civic offices with live data.
- **Complaint Reporting**: Report issues with structured AI categorization.
- **Document Guidance**: Accurate checklists for obtaining government IDs.
- **Scheme Recommender**: Personalized discovery of eligible welfare schemes.

## Technical Excellence

This project has been highly optimized for **Security**, **Performance**, and **Code Quality**:
- **Testing**: Robust test suite implemented using Vitest and React Testing Library.
- **Security**: Strict Content Security Policy (CSP), HTTP security headers, and Zod input validation on all API endpoints.
- **Performance**: Dynamic imports, React memoization, and Next.js App Router optimizations.
- **Accessibility**: ARIA compliant, keyboard navigable, and fully responsive.
- **PWA Ready**: Web App Manifest and theme configurations included.

## Setup Instructions

1. Clone the repository.
2. Install dependencies: \`npm install\`
3. Set your environment variables in \`.env.local\`:
   \`\`\`env
   GROQ_API_KEY=your_groq_api_key
   \`\`\`
4. Run the development server: \`npm run dev\`
5. To test, run: \`npm run test\`
